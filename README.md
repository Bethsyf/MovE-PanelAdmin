## Run Locally

Clone the project

```bash
  git clone https://github.com/Bethsyf/MovE-PanelAdmin/
```

Go to the project directory

```bash
  cd MovE-PanelAdmin
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
